Мова Python. Дані, типи даних та змінні

Основний процес, який реалізує будь-яка комп'ютерна програма - опрацювання даних. Ми знаємо, що дані в цілому - це структурована інформація, але ж не всі дані однакові з точки зору комп'ютерної програми. Вони можуть відрізнятися як за способом зберігання в оперативній пам'яті, так і за операціями, які над ними здійснює програма. І якщо у мові програмування Python різні дані зберігаються в пам'яті комп'ютера в однаковий спосіб (на відміну від, скажімо, мови C++), то розподіл у неї даних за їх типами для виконання операцій - надважливий момент, якій слід враховувати під час створення програми.

Подивіться на зображення вище: чому дорівнює чотири поділене на яблуко? Відповідь, вочевидь, не має сенсу, тому що об'єкт "яблуко" не може виступати у ролі дільника операції арифметичного ділення, бо така операція застосовується лише до чисел. Отже, дуже важливим є те, щоби дані у мові програмування відповідали операціям, які до них застосовуються.

Тип даних у мові програмування - набір правил, який описує певну величину, її властивості та дії, які можна виконати над нею.

У Python є чотири основні типи даних:

  • int - ціле число (integer)
  • float - дробове число
  • str - рядкова величина, текст (string)
  • bool - булева або логічна величина (boolean)

Створимо нашу першу програму. Вона буде виводити на екран дані всіх чотирьох типів. Для виведення даних на екран у Python існує функція print():

print(10)
print(3.14)
print("Hello, world")
print(True)

Результат:

print(10)
print(3.14)
print("Hello, world")
print(True)
