Аналіз рядів даних

У сучасному світі у багатьох галузях життєдіяльності активно використовується аналіз даних. Такий аналіз полягає у послідовному опрацюванні однотипових наборів даних шляхом застосування до них різноманітних методів, що дозволяють знайти певні закономірності, які допомагають дослідникам зробити необхідні висновки. Такі висновки дозволяють прийняти правильні рішення щодо певної проблематики з досліджуваної предметної області та навіть передбачити можливі майбутні наслідки.

Наприклад, аналіз зміни кількості представників різних вікових категорій населення держави допоможе передбачити зріст або спадання економіки, знаходження закономірностей у діях водіїв за кермом автомобілів - мінімізувати кількість аварій,  дослідження споживацьких звичок Інтернет-покупця - рекомендувати йому товар, який він охоче купить.

Науку, яка вивчає закономірності, отримані з даних, що описують масові явища, називають статистикою, а засоби опрацювання й аналізу даних у цій науці - статистичними методами.

Зрозуміло, що чим більше даних ми опрацьовуємо з метою отримання висновків -  тим більш ефективними будуть прийняті нами рішення. Але у більшості випадків зібрати й опрацювати всі дані з досліджуваної предметної області є вкрай складним завданням. Тому дослідники зосереджуються на створенні вибірки даних.

Вибірка - певна кількість однотипових наборів даних досліджуваної предметної області з усієї можливої множини таких наборів.

Можна сказати, що будь-яка вибірка має якісну характеристику. І дійсно, якісна вибірка повинна відповідати наступним критеріям:

  • масовість. Як вже було сказано, дуже часто збір та опрацювання всіх можливих наборів даних є вкрай складним завданням. Проте, слід дотримуватися простого правила: треба збирати та опрацьовувати максимально можливі об'єми даних, виходячи з власних технічних можливостей.
  • репрезентативність. Якщо, наприклад, ви досліджуєте який пристрій хоче придбати пересічний громадянин країни та при цьому охопите опитуванням лише топ-представників IT-галузі, то з великою долею вірогідності зробите висновок, що звичайний громадянин обов'язково придбає новий iPhone у топовій комплектації. Висновок буде неправильний - слід було охопити однакову кількість представників серед різноманітних категорій суспільства: за віком - від підлітків до пенсіонерів, за професією - від робочих спеціальностей до топ-менеджерів, за соціальним статусом, за статками тощо. Ідеальна репрезентативна вибірка має однакову кількість (але різний якісний показник!) представників всіх категорій респондентів, що мають відношення до досліджуваної предметної області.
  • релевантність. Слід запобігати потраплянню до вибірки будь-яких наборів даних, що не відносяться до досліджуваної предметної області. Процес відкидання "зайвих" даних з вибірки є важливим етапом підготовки даних до аналізу. Проте, стовідсоткове дотримання цього критерію під час опрацювання надвеликих об'ємів даних зазвичай є неможливим.

Дуже часто дані, отримані та попередньо оброблені для подальшого аналізу, записуються у табличній формі. Окремий рядок або стовпець з такої таблиці, який містить набір однотипових даних, що кількісно або якісно характеризують різні сутності однієї досліджуваної предметної області, називають рядом даних. У програмуванні такі ряди зручно зберігати й опрацьовувати за допомогою масивів.

Деякі статистичні методи

Середнє арифметичне значення. Цей метод покликаний дослідити, яким було би будь-яке значення у ряді, якщо всі значення цього ряду були би однаковими. Середнє арифметичне значення, як відомо, обчислюється як сума елементів ряду поділена на кількість цих елементів.

$${S}=\frac{\sum_{i=1}^{n}x_i}{n}$$

Уявимо, що ми маємо масив, у якому записані значення зросту учнів 10 класу:

Масив має 12 елементів, але може мати й набагато більше. У сучасному аналізі даних обробці підлягають масиви, що можуть містити мільйони елементів. Ми повинні вміти створювати програми опрацювання рядів будь-якого розміру.

# масив значень зросту учнів на Землі
h_earth = [170, 188, 192, 187, 190, 188, 171, 169, 167, 191, 188, 178]

Створимо програму знаходження середнього арифметичного значення:

# функція обчислення середнього арифметичного значення
def avg(arr):
    sum_val = 0 # сума значень
    # перебираємо всі наявні значення у масиві
    for i in arr:
    # сумуємо ці значення
        sum_val += i
    # повертаємо значення середнього арифметичного
    return sum_val

Викличемо цю функцію, передавши до неї масив:

print(f"Середній зріст учнів: {round(avg(h_earth),2)} см")

Результат:
Середній зріст учнів: 181.58 см

Отже, якщо би будь-який учень такого класу мав зріст 181.58 см, то й середнє арифметичне значень зросту учнів цього класу також дорівнювало би 181.58.

Середньоквадратове (стандартне) відхилення. Уявимо собі наступну ситуацію: маємо інопланетний клас з такими значеннями зросту учнів:

# масив значень зросту учнів на іншій планеті
h_alien = [0, 0, 0, 0, 0, 0, 363.16, 363.16, 363.16, 363.16, 363.16, 363.16]

Половина класу має зріст 0 см, половина - 363.16 см. Середнє арифметичне значення такого ряду також дорівнюватиме 181.58. Зробити якийсь висновок з отримання середнього арифметичного такого ряду буде вкрай важко.

Середньоквадратове (стандартне) відхилення демонструє, наскільки кожен елемент ряду відрізняється від середнього арифметичного значення такого ряду. Чим це значення ближче до нуля - тим більш однорідними є елементи ряду за своїми значеннями.

Середньоквадратове відхилення обчислюється за формулою:

$${S}=\sqrt{\sum_{i=1}^{n}\frac{(x_i-x_0)^2}{n}}$$

де \(x_i\) - наступний елемент ряду, а \(n\) - кількість елементів ряду.

Створимо функцію обчислення середньоквадратового відхилення:

# імпортуємо функцію знаходження квадратного кореня
from math import sqrt
# функція знаходження значення стандартного відхилення
def stdev(arr):
    # сума дорівнює 0
    sum_val = 0
    # знаходимо середнє арифметичне ряду
    avg_val = avg(arr)
    # перебираємо кожний елемент ряду
    for i in arr:
        # обчислюємо квадрат різності поточного елемента ряду
        # та середнього арифметичного значення
        current = (i - avg_val) ** 2
        # додаємо отримане значення до суми
        sum_val += current
    # повертаємо квадратний корінь суми,
    # поділеної на кількість елементів ряду
    return sqrt(sum_val / len(arr))

Викличемо функцію, передавши до неї дані з обидвох рядів:

print(f"Середньоквадратове відхилення землян: {round(stdev(h_earth),2)}")
print(f"Середньоквадратове відхилення інопланетян: {round(stdev(h_alien),2)}")

Результат:
Середньоквадратове відхилення землян: 9.36
Середньоквадратове відхилення інопланетян: 181.58

Для жителів Землі розбіжність у значеннях зросту (відхилення від середнього арифметичного) набагато менше, ніж для інопланетних учнів (бачимо, що стандартне відхилення для них дорівнює середньому арифметичному значенню).

Середньоквадратове відхилення зазвичай використовується для анализу ряду у парі із середнім арифметичним значенням. 

Мода ряду - значення, яке зустрічається в ряді найчастіше.

# функція, яка повертає моду
def mode(arr):
    # масив унікальних значень з ряду
    nums = []
    # масив кількості повторень для цих значень у ряді
    counts = []
    # проходимо циклом по ряду даних
    for i in range(len(arr)):
        # якщо елемент ряду вже є в масиві чисел,
        if arr[i] in nums:
            # збільшуємо значення кількості повторень для нього
            counts[nums.index(arr[i])] += 1
        else:
            # інакше додаємо значення з ряду до масиву чисел
            nums.append(arr[i])
            # ініціюємо лічильник повторень для цього числа
            counts.append(1)
    # припускаємо, що перша кількість повторень є найбільшою
    max_val = 0
    # перебираємо циклом всі наступні значення кількостей
    for i in range(1, len(counts)):
        # якщо якась кількість є більшою за наше припущення
        if counts[i] > counts[max_val]:
            # записуємо в змінну top його індекс
            max_val = i
    # повертаємо індекс числа, яке найбільш часто зустрічається
    return nums[max_val]

Викличемо функцію:

print(f"Мода ряду: {mode(h_earth)}")

Результат:
Мода ряду: 188

Медіаною називають значення, яке розташовано посередині відсортованого ряду. Уявимо, що маємо ряд, який описує зарплатню на підприємстві:

salary = [12000, 8000, 270000, 9000, 17000, 32000]

Середня зарплатня на підприємстві:

print(f"Середня зарплатня: {avg(salary)} грн")

Результат:
Середня зарплатня: 58000.0 грн

Зарплатня у 270000, яка, вочевидь, належить керівникові, чинить дуже великий вплив на середнє арифметичне значення, тому у даному випадку застосування статистичного метода середнього арифметичного значення не є доцільним. Натомість, дослідимо медіанне значення. Створимо функцію, яка буде повертати медіану ряду:

# функція, яка повертає медіану ряду
def median(arr):
    # створюємо копію масиву  
    copied = arr.copy()
    # сортуємо копію масиву
    copied.sort()
    # повертаємо середній елемент масиву
    return copied[(len(copied) - 1) // 2]

Застосуємо функцію до нашого ряду:

print(f"Медіанна зарплатня: {median(salary)} грн")

Результат:
Медіанна зарплатня: 12000 грн

Таке значення є більш показовим у якості "серединної" заробітної платні.

Як бачимо, використання статистичних методів під час аналізу даних доволі сильно залежить від досліджуваної предметної області.
