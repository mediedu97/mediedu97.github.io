Що таке програмування. Мови програмування

Комп'ютер є електронною машиною, яка вміє виконувати арифметичні та логічні операції над даними з надвисокою швидкістю згідно зі списком послідовних інструкцій, які містяться в її оперативній пам'яті. Такий список називають комп'ютерною програмою.
Проблема у тому, що комп'ютер розуміє лише інструкції, записані двійковим кодом, тому пряме спілкування з ним з метою надання йому інструкцій для виконання є вкрай ускладненою процедурою для людей. Саме ця проблема призвела до появи великої кількості мов програмування.

Мова програмування - набір правил для створення спеціального тексту, який

  • містить послідовність інструкцій для виконання комп'ютером
  • зазвичай дуже схожий на записи англійською мовою
  • призначений для швидкого перетворення самими комп'ютером на двійковий код, який він виконуватиме

Цей спеціальний текст також називають комп'ютерною програмою, написаною на мові програмування.

Приклад комп'ютерної програми:

for i in range(10):
    if i % 2 == 0:
        print(f"{i} - парне число")

Ми бачимо багато англійських слів та інших символів (знаки пунктуації, математичні знаки та символи й числа, що утворюють вирази).

Будь-яка мова програмування складається з трьох компонентів:

  • алфавіт - набір всіх можливих символів, які входять до мови
  • синтаксис - правила запису слів, виразів,
  • семантика - значення, яке утворюють слова, вирази та їх комбінації

У сучасному світі за допомогою мов програмування, які, як бачимо, схожі на людську набагато більше, ніж двійковий код, програмісти створюють майже всі комп'ютерні та мобільні застосунки.

Мови програмування поділяють на різні категорії за способом їх перетворення на двійковий код:

  • інтерпретовані - програма перетворюється на двійковий код під час її виконання (інтерпретується) спеціальною програмою-інтерпретатором, яка повинна бути встановлена на комп'ютері. До таких мов належать Python, JavaScript та інші. Така програма може бути виконаною на будь-якому пристрої, на якому є у наявності відповідний інтерпретатор.
  • компільовані - програма перетворюється на двійковий код перед виконанням її комп'ютером (компілюється) спеціальною програмою, яку називають компілятор. Для виконання компільованої програми не потрібно встановлювати інші спеціальні програми, адже вона вже записана у двійковому коді. Проте, слід розуміти, що різні процесори мають свої власні набори інструкцій (архітектуру), тому програма, скомпільована для розуміння одним типом процесорів, край можливо, не зможе бути виконана процесором іншої архітектури. Прикладом компільованих мов програмування є C, C++, Rust, Pascal.
  • гібридні або проміжні - компільовані не у двійковий, а у спеціальний проміжний або байт-код. Такий байт-код потім у реальному часі перетворюється на двійковий  за допомогою віртуальної машини - спеціальної програми, яка повинна бути встановлена на комп'ютері. Гібридні програми виконуються швидше, ніж інтерпретовані, але повільніше за компільовані. Серед мов програмування проміжного типу можна виділити Java та C#.

З алфавіту мови Python створюються лексеми - одиниці мови, які аналізуються програмою-інтерпретатором і які у випадку правильного їх розуміння перетворюються  у ході інтерпретації на так звані токени - проміжні мовні конструкції, призначені для подальшого перетворення програмного коду на двійковий. Лексема несе змістовне навантаження в залежності від її виду.

Розрізняють такі види лексем:

  • ключові слова (keywords). Це окремі команди мови програмування, які реалізують виконання базових алгоритмічних конструкцій (циклів, умов) та інших стандартних для мови програмування дій
  • ідентифікатори (identifiers). Такі лексеми як правило містять назви змінних, функцій, класів та об'єктів
  • літерали (literals). Літералом називають конкретне значення
  • оператори (operators). Наприклад, додавання, віднімання, порівняння значень тощо. Сутності, над якими оператори здійснюють операції (наприклад, ідентифікатори або літерали), називають операндами. У виразі 2+3 операндами є значення 2 і 3 а оператором є +.
  • знаки пунктуації

Записані у певній послідовності лексеми створюють текст комп'ютерної програми або, як говорять програмісти, програмний код чи просто код. Цей код розуміють всі програмісти, які знайомі з відповідною мовою програмування. Зрозуміло, що не слід плутати програмний код з двійковим кодом, який є зрозумілим саме для комп'ютера. Двійковий код є кінцевим результатом компіляції або інтерпретації тексту комп'ютерної програми.

Отже, програмування - це процес створення алгоритму виконання певних задач і його текстового запису однією з мов.

Всього на даний момент існує більше 300 мов програмування. Звідки взялася така кількість? Існує багато причин, основні з яких - різноманітне цільове призначення окремих мов та бажання різних IT-корпорацій та суспільств заохотити програмістів розвивати саме їх програмні продукти. Серед багатьох прикладів - мова С# створена та підтримується корпорацією Microsoft, Google створили та підтримують мови Dart та Golang, Mozilla є автором і власником прав на мову Rust.

За цільовим призначенням мови програмування історично розрізняють на загального призначення та спеціалізовані, де загальні мови програмування були покликані вирішувати широке коло задач, у той час як спеціалізовані були зосереджені на вирішені проблем специфічних. У сучасному світі майже всі мови програмування є загальними та можуть застосовуватися для будь-яких задач, хоча деякі й досі продовжують домінувати саме у своїй історичній галузі, для якої вони були розроблені. Наприклад:

  • JavaScript - створення динамічних веб-сторінок
  • С++ - програмування високонавантажених та швидкісних систем
  • С - програмування операційних систем та мікроконтролерів
  • С# - програмування бізнес-застосунків для операційної системи Windows
  • Java - програмування Android-застосунків та вбудованих систем
  • Python - аналіз даних, машинне навчання, наукові обчислення

Будь-яка мова має свої переваги та недоліки. Серед основних факторів під час вибору мови під окрему задачу слід виділити

  • швидкодію виконання програми (серед лідерів - C, C++, Rust)
  • лаконічність, простота, швидкість розробки та налагодження програмного коду (Python, JavaScript)
  • збалансованість факторів, поданих вище (C#, Kotlin, Dart)

Ми обрали мову Python через простоту її використання, лаконічність синтаксису та швидкість написання програмного коду.
