Що таке програмування. Мови програмування

Комп'ютер є електронною машиною, яка вміє виконувати арифметичні та логічні операції над даними з надвисокою швидкістю згідно зі списком послідовних інструкцій, які містяться в її оперативній пам'яті. Такий список називають комп'ютерною програмою.
Проблема у тому, що комп'ютер розуміє лише інструкції, записані двійковим кодом, тому пряме спілкування з ним з метою надання йому інструкцій для виконання вкрай ускладненою процедурою для людей.

Мова програмування - набір правил для створення спеціального тексту, який

  • містить послідовність інструкцій для виконання комп'ютером
  • зазвичай дуже схожий на записи англійською мовою
  • призначений для того, щоб його можна було б легко перетворити на двійковий код

Цей спеціальний текст також називають комп'ютерною програмою, написаною на мові програмування.

Приклад комп'ютерної програми:

for i in range(10):
    if i % 2 == 0:
        print(f"{i} - парне число")

Ми бачимо багато англійських слів та інших символів (знаки пунктуації, математичні знаки та символи й числа, що утворюють вирази). Будь-яка мова програмування складається з трьох компонентів:

  • алфавіт - набір всіх можливих символів, які входять до мови
  • синтаксис - правила запису слів, виразів,
  • семантика - значення, яке утворюють слова, вирази та їх комбінації

У сучасному світі за допомогою мов програмування, які, як бачимо, схожі на людську набагато більше, ніж двійковий код, програмісти створюють майже всі комп'ютерні та мобільні застосунки.

Мови програмування поділяють на різні категорії за способом їх перетворення на двійковий код:

  • інтерпретовані - програма перетворюється на двійковий код під час її виконання (інтерпретується) спеціальною програмою-інтерпретатором, яка повинна бути встановлена на комп'ютері. До таких мов належать Python, JavaScript та інші. Така програма може бути виконаною на будь-якому пристрої, де встановлений відповідний інтерпретатор.
  • компільовані - програма перетворюється на двійковий код перед виконанням її комп'ютером (компілюється) спеціальною програмою, яку називають компілятор. Для виконання компільованої програми не потрібно встановлювати інші спеціальні програми, адже вона вже записана у двійковому коді. Проте, слід розуміти, що різні процесори мають свої власні набори інструкцій (архітектуру), тому програма, скомпільована для розуміння одним типом процесорів, край можливо, не зможе бути виконана іншим. Прикладом компільованих мов програмування є C, C++, Rust, Pascal.
  • гібридні - компільовані не у двійковий, а у спеціальний проміжний або байт-код. Такий байт-код потім у реальному часі перетворюється на двійковий  за допомогою віртуальної машини - спеціальної програми, яка повинна бути встановлена на комп'ютері. Гібридні програми виконуються швидше, ніж інтерпретовані, але повільніше за компільовані. Серед мов програмування проміжного типу можна виділити Java та C#.
